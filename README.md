## Installation and Setup Instruction
#### Run the following commands in your terminal

`git clone https://github.com/Gaursafal/Library-Management`

`cd Frontend`

`npm install`

`npm start`

`cd Backend`

`nodemon/node server.js`

## Libraries installed

- Material UI
- Axios
- React router dom
- Styled components
- Express-js
- Cors
- dotenv
- nodemon
